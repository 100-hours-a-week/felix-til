#### 브라우저 렌더링 과정
```
DOM - Document Object Model 
CSSOM - CSS Object Model
Render Tree - DOM + CSSOM (보여지는 것)
layout - 각 노드의 크기 및 위치 계산
paint - RenderTree를 실제 픽셀로 변환
composite - 모든 레이어를 하나의 최종이미지로 합성
```
