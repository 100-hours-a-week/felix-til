### 🚀 Today I Learned
#### 브라우저 스토리지
```
로컬 스토리지
- 브라우저 탭을 닫아도 데이터 유지
세션 스토리지
- 브라우저 탭을 닫으면 데이터 자동 삭제

위의 두 브라우저 스토리지의 공통점
- 자바스크립트로 접근 가능
- 서버로 자동 전송되지 않음
```

