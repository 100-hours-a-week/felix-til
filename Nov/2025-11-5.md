### 🚀 Today I Learned
#### VPN(가상사설망)
```
사용자가 공용 네트워크를 통해 안전하게 사설 네트워크에 접속할 수 있도록 해주는 기술

터널링 
- 사용자 기기와 VPN 서버 사이에 가상의 전용 통로를 만듦
- 사용자의 데이터 패킷은 터널로 진입하기 위해 VPN 프토토콜을 사용하여 캡슐화
- 캡슐화된 패킷은 일반 인터넷망을 통과하여 VPN 서버로 전송

암호화
- 터널을 통과하는 데이터를 암호화하여 중간에 데이터를 가로채더라도 내용을 해독할 수 없도록 함

VPN은 서버를 경유하므로 인터넷 속도가 약간 느려질 수 있음
```
